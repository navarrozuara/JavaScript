El datepicker está vinculado a un campo de entrada de formulario estándar. Se centra en la entrada (clic, o utilice la tecla tab) para abrir un calendario interactivo en una pequeña superposición. Para cerrar el calendario elija una fecha, haga clic en otra parte de la página (desenfoque la entrada) o pulse la tecla Esc. Si se elige una fecha, se muestra como valor de entrada.

Código jQuery

$("#datepicker").datepicker({
    dateFormat: "dd-mm-yy",
    minDate: new Date(),
    dayNames: [ "Domingo", "Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado" ],
    dayNamesMin: [ "Dom", "Lun", "Mar", "Mie", "Jue", "Vie", "Sab" ],
    monthNames: [ "Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", "Julio", "Agosto", "Septiembre", "Octubre", "Noviembre", "Diciembre" ],
    monthNamesShort: [ "Ene", "Feb", "Mar", "Abr", "May", "Jun", "Jul", "Ago", "Sep", "Oct", "Nov", "Dic" ],
    showAnim: "drop"
});

This is an animated dialog which is useful for displaying information. The dialog window can be moved, resized and closed with the 'x' icon.

La ventana de diálogo básico es una superposición que se coloca dentro de la vista y está protegido del contenido de la página (como elementos escogidos) que brilla a través de un iframe. Tiene una barra de título y un área de contenido, y se puede mover, cambiar de tamaño y se cierra con el icono 'x' por defecto.

Código jQuery

$("#dialog").dialog({
    autoOpen: false,
    show: {
        effect: "blind",
        duration: 1000
    },
    hide: {
        effect: "explode",
        duration: 1000
    },
    title: "Diálogo básico",
    width: 400,
    buttons: [
        {
            text: "Ok",
            click: function() {
                $(this).dialog("close");
            }
        },
        {
            text: "Cancel",
            click: function() {
                $(this).dialog("close");
            }
        }
    ]
});
$("#opener").on('click', function() {
    $("#dialog").dialog("open");
});

El tooltip puede acoplarse a cualquier elemento. Al pasar por encima del elemento con el cursor del ratón, se muestra el atributo title en una pequeña caja al lado del elemento, al igual que un texto de ayuda nativa.

Código jQuery

$("#usuario").tooltip({
    show: { 
        effect: "blind", 
        duration: 1000 
    }
});
