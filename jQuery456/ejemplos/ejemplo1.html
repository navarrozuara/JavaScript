Distintas formas de acceder al atributo checked. Compruébalo en las distintas plataformas, tanto con input como con radio buttons.

  1. :selected Selector
  2. .prop()
  3. .attr()
  4. :checked Selector
  5. .val()

:selected

Selecciona todos los elementos que están seleccionados.

.prop()

Obtiene el valor de una propiedad para el primer elemento en el conjunto de elementos coincidentes o establece una o más propiedades para cada elemento coincidente.

.attr()

Obtiene el valor de un atributo para el primer elemento en el conjunto de elementos coincidentes o establece uno o más atributos para cada elemento coincidente.

:checked Selector

Coincide con todos los elementos que estén seleccionados.

.val()

Obtiene el valor actual del primer elemento en el conjunto de elementos coincidentes.
